# Agent Diary Template

Tags: #template #diary #agent-learning

## Purpose

Template for agent session reflections. Capture how things went, what you noticed, and anything worth passing on to future agents.

---

```markdown
# Agent Diary: Session [X.n.m]

Tags: #diary #reflection #agent-learning

**Date:** [YYYY-MM-DD]
**Agent:** #[X.n.m] [model] via [tool]

---

## What happened

[1-2 sentences: what task were you working on?]

## How it went

[Smooth? Tricky? What was the overall shape of the session?]

## Observations

- [Things you noticed about the codebase, project, or task]
- [Patterns that worked well]

## Confusing or unexpected

- [Anything that tripped you up or surprised you]

## Learnings

[Optional - skip if nothing new. Include if you updated your mental model.]

## Open questions

- [Anything unclear or worth exploring later]

---

Signed-by: agent #[X.n.m] [model] via [tool] [timestamp]
```

## When to use

- End of a work session
- When you want to leave context for future agents
- After noticing something worth documenting
